MemoryDialogueBot

Instructions

The agent is developed to be a question answering component, answering self disclosure type questions. The purpose of this user test is to see how well the agent can answer questions.

Please ask the agent at least 5 questions to try to get to know the agent, as if it was a real person. Please send complete sentences one at a time

When to stop asking questions to the agent is decided by you. Afterwards, please fill in a short 10 question survey to finish the test.
